Beacons Module

Description

StreetHawk's Beacon module lets you to run campaigns based on user's presence inside a beacon's region.

Accessing the Beacons Module

To access this module from JavaScript, you would do the following:

var shBeacon = require("com.streethawk.Beacons");

The shBeacon variable is a reference to the Module object.

Reference

https://streethawk.freshdesk.com/solution/articles/12000000943-beacon

beacons.function

  • Enter beacon

    shBeacon.shEnterBeacon();

  • Exit beacon

    shBeacon.shExitBeacon();

beacons.property

  • Set or get enable location by default

    shBeacon.defaultLocationService = false; var isEnabledByDefault = shBeacon.defaultLocationService;

  • Set or get currently enable location

    shBeacon.locationEnabled = true; var isEnabled = shBeacon.locationEnabled;

Usage

  • Delay asking for Permission

By default StreetHawk library request location permission when call "streethawkinit". In case you may wish to delay asking the user for location permission, you can add the code be as shown below.

var shBeacon = require('com.streethawk.Beacons');
//optional to delay ask for permission
shBeacon.defaultLocationService = false;
//do streethawkinit and other things.
function buttonLocationEnableClicked(e) {
    //Enable location
    shBeacon.locationEnabled = true;
}
